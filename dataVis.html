A collection of D3.js dashboards and diagrams made for Harvard's data visualization course.

D3 DASHBOARD: VISUALIZING COVID-19 IN THE US

For my assignment on visualizing Covid-19's impact on US states, I created a dashboard displaying Covid-19 statistics for each state over time on a continuous color scale. The dashboard uses a brush and zoom timeline to filter the geomap, table, and bar graph data by date range. The user also has the option to filter by absolute or population relative cases or deaths, and all graphs will transition upon doing so. Displayed is an interactive geomap, the ten highest and ten lowest states for each filtered outcome and the raw-number data table on a carousel opposite the geomap.

SPLIT MATRIX DIAGRAM

This visualization uses two historical datasets and custom SVG shapes to generate an interactive split-matrix diagram displaying Florentine family relationships and rankings in response to user selections.

SELECTORS AND TRANSITIONS

This visualization uses World Cup data from 1930-2014 to create an interactive line chart and info card based on the user's selected metrics. Graphs transition between metric selections.

BRUSH AND ZOOM

This visualization uses UK household food and drink expenditure data to create a stacked area chart and timeline the user can brush and zoom to filter by time.
